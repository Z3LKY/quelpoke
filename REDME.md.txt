# Application QUELPOKE

Cette application vous permet de découvrir quel Pokémon vous êtes en fonction de votre prénom. Chaque prénom est associé à un Pokémon spécifique. Si vous entrez un nouveau prénom, l'application vous attribuera un nouveau Pokémon et enregistrera ce prénom.

## Fonctionnalités

- Entrer un prénom pour découvrir quel Pokémon vous êtes.
- Générer un Pokémon différent lorsque le prénom change.
- Stocker les prénoms et leurs Pokémon associés.

## Comment ça fonctionne ?

1. **Lancer l'application** : Lorsque vous lancez l'application, elle vous demandera de saisir votre prénom.
2. **Entrer un prénom** : Saisissez votre prénom, et l'application vous associera un Pokémon aléatoire.
3. **Stockage des données** : L'application stockera le prénom et le Pokémon associé dans une base de données locale.
4. **Changement de prénom** : Si vous entrez un nouveau prénom, l'application générera un nouveau Pokémon et mettra à jour la base de données avec ce nouveau prénom et Pokémon.

## Technologies utilisées

- Langage de programmation : Python (ou un autre langage approprié).
- Interface utilisateur : Interface en ligne de commande ou graphique (selon vos préférences).
- Stockage des données : Fichier texte, CSV, ou base de données SQLite.
